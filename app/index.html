General Data Structures

A quick tour of the most common structures—with visuals.

Why Data Structures Matter

A data structure is just a way to organize data so common tasks are fast and simple. The “right” choice depends on what you do most: look things up, insert, delete, or keep order.

Data Structure Gallery

Arrays

Fast index access • good cache locality

Store items back-to-back. Great for reading by position. Inserts in the middle are slower (need to shift).

Linked Lists

Cheap inserts/removals with a cursor

Each node points to the next. No index access; but adding/removing near a node is quick.
