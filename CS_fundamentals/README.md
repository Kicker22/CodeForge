# CS Fundamentals

This directory contains fundamental computer science concepts and programming principles, organized into two main categories:

## 📁 Directory Structure

### 1. Programming Concepts
Located in `programming_concepts/`

Python-specific programming concepts and language features:

- **arrays/** - Working with arrays/lists in Python: creation, manipulation, iteration, and real-world examples
- **functions/** - Function definitions, parameters, return values, *args, **kwargs, and best practices
- **methods/** - Object-oriented methods, class definitions, instance methods, and inheritance
- **objects/** - Object-oriented programming: classes, instances, attributes, and inheritance

### 2. Data Representation
Located in `data_representation/`

Low-level computer science concepts explaining how data is stored in binary:

- **bits_and_bytes/** - Binary number system, base conversions, and the building blocks of computer data
- **integers/** - How whole numbers are represented in binary (signed vs unsigned, two's complement)
- **floats/** - IEEE 754 floating-point representation for decimal numbers and scientific notation
- **characters/** - Character encoding (ASCII, Unicode) and how text is represented in binary

## 🎯 Purpose

This directory serves as:
- A learning resource for fundamental CS concepts
- Reference material for understanding how computers work at a low level
- Practical Python examples for common programming patterns

## 📚 Learning Path

**Recommended order for beginners:**

1. Start with **Programming Concepts** to understand Python basics
   - arrays → functions → methods → objects
   
2. Then dive into **Data Representation** to understand how computers work
   - bits_and_bytes → integers → characters → floats

## 🔗 Related Directories

- `data_structures/` - More advanced data structures (linked lists, trees, graphs)
- `algorithms/` - Algorithm implementations (sorting, searching, graph algorithms)
- `challenges/` - Practice problems to apply these concepts
